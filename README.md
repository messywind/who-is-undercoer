# 谁是卧底 · 本地传手机版

一个用 Vue3 + Vite 写的本地「谁是卧底」游戏，针对手机（iPhone）做了适配。一台手机即可游玩：房主设置人数与自定义词汇，然后把手机依次传给每位玩家，长按查看各自的词语。

## 玩法

1. **设置**：选择总人数、卧底人数，填写「平民词汇」和「卧底词汇」，并输入每位玩家的 ID。
2. **开始分配身份**：系统随机分配身份。
3. **看身份**：把手机传给玩家，点开自己的名字，**长按**卡片查看词语，松开隐藏。
4. 所有人查看完即可开始线下讨论；游戏结束后可「公布所有身份」。
5. 「重发」可用同一批玩家重新随机分配。

## 本地开发

```bash
npm install
npm run dev      # 本地预览（手机和电脑在同一 Wi-Fi 下，用电脑局域网 IP 访问）
npm run build    # 打包到 dist/
```

## 部署到 GitHub Pages

1. 在 GitHub 新建一个仓库（例如 `sswd`），把本目录推送上去（默认分支 `main`）。
2. 仓库 **Settings → Pages → Build and deployment → Source** 选择 **GitHub Actions**。
3. 推送到 `main` 后，`.github/workflows/deploy.yml` 会自动构建并发布。
4. 完成后访问 `https://<你的用户名>.github.io/<仓库名>/`。

> 已设置 `base: './'`（相对路径），无论部署在哪个子路径都能正常加载，无需修改配置。
